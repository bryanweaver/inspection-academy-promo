# Documents Folder

Place your official documents in this folder:

## Required Files

1. **official-rules.pdf** - The official scholarship rules document
2. **privacy-policy.pdf** (optional) - If you want to host privacy policy here instead of linking to main site

## How to Add Files

1. Save your Official Rules document as a PDF
2. Name it exactly: `official-rules.pdf`
3. Drop it into this `docs` folder
4. Commit and push to GitHub

The rules will be accessible at:
`https://promo.theinspectionacademy.com/docs/official-rules.pdf`
